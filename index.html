공부 메모


태그 용어정리

  • li: 리스트
  • h숫자: 제목
  • ol: ordered list
  • ul: unordered list
  • head: 전체 속성 또는 규칙을 정의하는 부분
  • body: 페이지 본문 내용
  • meta: 해당 웹페이지를 어떤 규칙에 의해서 불러올지 정의
  • !doctype: 해당 문서가 어떤 타입인지 정의
  • html: 전체 문서의 처음과 끝에 기재함
  • src: source
  • a: anchor. 링크를 의미함
  • href: h(하이퍼텍스트) + ref(참조)
  • target="_blank": 새탭에서 열기
